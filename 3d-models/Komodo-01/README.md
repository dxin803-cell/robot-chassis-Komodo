# Komodo-01 3D Models

Comprehensive 3D model collection for the Komodo-01 mid-range robot chassis.

## 📦 What's Included

- **chassis-assembly.stl/step/obj** - Complete chassis assembly
- **motor-bracket.stl** - Motor mounting bracket (heavy-duty)
- **wheel-hub.stl** - Wheel hub component
- **caster-wheel-mount.stl** - Caster wheel bracket
- **battery-tray.stl** - Dual battery mounting tray
- **controller-mount.stl** - Motor controller bracket
- **sensor-hub-mount.stl** - Sensor board mounting plate
- **cable-management.stl** - Cable routing and management
- **weatherproof-cover.stl** - Optional protective cover

## 📐 Specifications

- **Scale**: 1:1 (Full size)
- **Units**: Millimeters (mm)
- **Material**: Aluminum alloy (suggested for metal parts)
- **Tolerance**: ±0.3mm
- **Weatherproofing**: IP67 compatible design

## 🔧 Assembly Quick Reference

1. Start with reinforced chassis base
2. Mount heavy-duty motor brackets (2×)
3. Install 24V DC motors in brackets
4. Attach larger wheels to motor shafts
5. Mount dual motor controllers
6. Install battery trays (dual capacity)
7. Add sensor hub on top plate
8. Secure weather seals around connectors
9. Install caster wheel assembly

See [Assembly Guide](../../docs/assembly-guide.md) for detailed instructions.

## 🎨 Visualization Tips

### In GitHub
1. Click any `.stl` file above
2. Model loads automatically
3. Use mouse to rotate/zoom
4. Click download to get the file

### In CAD Software
1. Download `.step` file
2. Open in FreeCAD, Fusion 360, SolidWorks
3. Explore assembly structure
4. Modify as needed
5. Export to your format

### For 3D Printing
1. Download `.stl` file
2. Open in Cura, PrusaSlicer, Prusament
3. Scale to 100%
4. Print with 25-40% infill
5. Use support material for overhangs
6. Expected print time: 8-20 hours per major component
