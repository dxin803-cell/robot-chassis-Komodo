# Komodo-ZERO 3D Models

Comprehensive 3D model collection for the Komodo-ZERO compact robot chassis.

## 📦 What's Included

- **chassis-assembly.stl/step/obj** - Complete chassis assembly
- **motor-bracket.stl** - Motor mounting bracket
- **wheel-hub.stl** - Wheel hub component
- **caster-wheel-mount.stl** - Caster wheel bracket
- **battery-tray.stl** - Battery mounting tray
- **controller-mount.stl** - Motor controller bracket
- **cable-guide.stl** - Cable routing guides

## 📐 Specifications

- **Scale**: 1:1 (Full size)
- **Units**: Millimeters (mm)
- **Material**: Aluminum (suggested for metal parts)
- **Tolerance**: ±0.2mm

## 🔧 Assembly Quick Reference

1. Start with chassis-assembly base
2. Mount motor brackets (2×)
3. Install motors in brackets
4. Attach wheels to motor shafts
5. Mount controller on designated bracket
6. Install battery in tray
7. Secure caster wheel at rear

See [Assembly Guide](../../docs/assembly-guide.md) for detailed instructions.

## 🎨 Visualization Tips

### In GitHub
1. Click any `.stl` file above
2. Model loads automatically
3. Use mouse to rotate/zoom
4. Click download to get the file

### In CAD Software
1. Download `.step` file
2. Open in FreeCAD, Fusion 360, etc.
3. Explore assembly structure
4. Modify as needed

### For 3D Printing
1. Download `.stl` file
2. Open in Cura, PrusaSlicer, etc.
3. Scale to 100%
4. Print with 20-30% infill
5. Use support material as needed
