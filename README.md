# JCROBOT Komodo Series Robot Chassis

Welcome to the official repository for the **JCROBOT Komodo Series** robot chassis platform. This repository contains comprehensive documentation, 3D models, technical specifications, and setup guides for all Komodo chassis variants.

## 📋 Overview

The Komodo Series is a modular, scalable robot chassis platform designed for various applications including autonomous navigation, industrial automation, research, and educational purposes. Each variant is optimized for different payload capacities, speed requirements, and terrain conditions.

## 🤖 Product Variants

| Model | Payload | Max Speed | Terrain | Status |
|-------|---------|-----------|---------|--------|
| [Komodo-ZERO](#komodo-zero) | 5 kg | 1.5 m/s | Indoor | ✅ Available |
| [Komodo-01](#komodo-01) | 15 kg | 1.2 m/s | Indoor/Outdoor | ✅ Available |
| [Komodo-02](#komodo-02) | 25 kg | 1.0 m/s | Outdoor | ✅ Available |
| Komodo-03 to Komodo-15 | Varies | Varies | Multi-terrain | 🔄 Coming Soon |
