# Komodo-02 3D Models

Comprehensive 3D model collection for the Komodo-02 heavy-duty industrial robot chassis.

## 📦 What's Included

- **chassis-assembly.stl/step/obj** - Complete chassis assembly (reinforced)
- **motor-bracket.stl** - Heavy-duty motor mounting bracket
- **wheel-hub.stl** - Industrial-grade wheel hub component
- **caster-wheel-mount.stl** - Heavy-duty caster wheel bracket
- **battery-tray-dual.stl** - Dual large battery mounting tray
- **controller-mount.stl** - Dual motor controller bracket
- **sensor-hub-mount.stl** - Multi-sensor integration plate
- **cable-management.stl** - Industrial cable routing system
- **weatherproof-enclosure.stl** - Full IP67 protective enclosure
- **payload-plate.stl** - Heavy-duty payload mounting plate
- **bumper-assembly.stl** - Full perimeter bumper protection

## 📐 Specifications

- **Scale**: 1:1 (Full size)
- **Units**: Millimeters (mm)
- **Material**: Aluminum alloy 6061-T6 (recommended)
- **Tolerance**: ±0.5mm
- **Weatherproofing**: Full IP67 design
- **Load Rating**: 25 kg payload

## 🔧 Assembly Quick Reference

1. Start with reinforced aluminum chassis base
2. Mount industrial-grade motor brackets (2×)
3. Install high-torque 24V DC motors
4. Attach all-terrain wheels to motor shafts
5. Mount dual motor controllers with redundancy
6. Install dual battery trays with isolation
7. Add multi-sensor hub on top plate
8. Install full weatherproofing seals
9. Mount payload plate for cargo
10. Attach perimeter bumpers for obstacle protection
11. Secure caster wheel assembly

See [Assembly Guide](../../docs/assembly-guide.md) for detailed step-by-step instructions.

## 🎨 Visualization Tips

### In GitHub
1. Click any `.stl` file above
2. Model loads automatically in browser
3. Use mouse to rotate/zoom/pan
4. Right-click to download

### In Professional CAD Software
1. Download `.step` file (full assembly)
2. Open in SolidWorks, Fusion 360, or FreeCAD
3. Explore modular assembly structure
4. Modify individual components
5. Generate manufacturing drawings
6. Create custom configurations

### For Industrial 3D Printing
1. Download `.stl` files
2. Open in professional slicing software
3. Scale to 100% (full scale)
4. Print with 35-50% infill (high strength)
5. Use metal-filled filament for durability
6. Post-process if needed (annealing, surface treatment)
7. Expected print time: 20-40+ hours per major component
8. Alternative: Send `.step` to professional SLS/DMLS service
