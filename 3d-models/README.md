## 📐 File Formats Explained

### .STL (Stereolithography)
- **Best for**: 3D printing, visualization, slicing software
- **Viewer**: Thingiverse, Printables, Fusion 360, FreeCAD
- **Properties**: Binary format, good for 3D printing

### .STEP (Standard for the Exchange of Product model data)
- **Best for**: CAD editing, assembly, parametric modeling
- **Software**: Fusion 360, SolidWorks, FreeCAD, OnShape
- **Properties**: Industry standard CAD format

### .OBJ (Wavefront Object)
- **Best for**: Web viewers, game engines, rendering
- **Viewer**: Sketchfab, Three.js, Babylon.js, Unity
- **Properties**: Text format, supports colors and textures

## 📖 How to Use These Models

### For 3D Printing
1. Download `.stl` file for your component
2. Open in slicing software (PrusaSlicer, Cura, etc.)
3. Adjust print settings
4. Print and assemble

### For CAD Modification
1. Download `.step` file
2. Open in CAD software (Fusion 360, FreeCAD, etc.)
3. Modify as needed
4. Export to desired format

### For Web Viewing
1. Download `.obj` file
2. Upload to Sketchfab or similar viewer
3. Share online
4. Collaborate with others

## 🔍 Model Details
